# Sudong (树洞)

一个安全的匿名聊天空间，基于 React + TypeScript + Vite + TailwindCSS 构建的 PWA 应用。

🔗 在线地址：**[williamzlj.github.io/sudong](https://williamzlj.github.io/sudong)**

## 功能特性

- AI 机器人对话，倾诉你的心声
- 聊天记录管理（搜索、筛选、置顶、备注、导出）
- 待办事项管理（待处理/已完成/参考）
- 多语言支持（中文、English、日本語、한국어）
- 暗色/亮色模式切换
- PWA 离线支持，可安装到桌面/手机
- 纯本地存储，数据保存在浏览器 IndexedDB，不上传服务器

## 技术栈

React 18 | TypeScript | Vite 5 | TailwindCSS 3 | i18next | IndexedDB | PWA
